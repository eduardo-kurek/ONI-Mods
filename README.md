# Eduardo Kurek - Oxygen Not Included MODS
My mods for ONI.

# Mod List

- **Circuit Not Included**: A mod to create custom circuits and provides the capability to customize logic ports.

# Report a bug
If you find a bug in a mod, please feel free to open an issue specifying the bug; also, inform the logs provided by the game if it was a crash.
You can suggest a new feature too, just open the issue and tell how you would like the feature to be. Also inform the mod name on the issue.

# How to contribute
To contribute, you should comment on a certain issue that you want to work on.
Then I will assign you to the issue, and you can send your pull requests. Feel free to ask for help.
If a long time has passed (sort of a month) and you do not manifest activity on the issue, I may assign other people if someone wants to.
I will consider the pull request of the assignee; if there is no assignee, the first pull request will be considered.
